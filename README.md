今までは2012年の新卒から電話営業に従事し、インターネット回線、生命保険の営業を行って参りました。

<BR>その際に、わかりやすいHPも多いのですが、途中で作成を終わった感じや更新をあまりしてないHPも多く、もったいない気持ちがありました。

<BR>将来的には、介護×IT、農業×IT、飲食×ITのように色々な分野でITの技術を発展させて私もその職業を体験しながらITでお手伝いになれたらと考えております。
